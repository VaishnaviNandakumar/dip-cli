## DIP-CLI
A cli tool to implement image processing filters on an image or a sample matrix.

##Introduction to digital image processing
##Filters available
* Spatial Domain
  - Laplacian Filter
  - Sobel Filter
  - Prewitt Filter
  - Statistical Filters
    - Mean
    - Median
    - Max
    - Min
* Frequency Domain
  - Ideal Filter
  - Gaussian Filter
  - Butterworth Filter
